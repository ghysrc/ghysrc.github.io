《非暴力沟通》- 好好说话

男：你无情，你残酷，你无理取闹！

女：那你就不无情，不残酷，不无理取闹？

男：我哪里无情，哪里残酷，哪里无理取闹？

女：你哪里不无情，哪里不残酷，哪里不无理取闹？

虽然这是影视剧里面矫情的台词对话，但现实中我们也会以类似的方式进行沟通。女孩说“你心里就没我！”，男孩就条件反射一般“怎么就没你了？”，接着就是“哪里有我了？”，“哪里没你了？”......

与书名“非暴力沟通”相对的，上面这种就是暴力沟通的一种。言语上的指责、嘲讽、否定、说教，以及任意打断、拒不回应、随意出口的评价和结论，都是暴力沟通。它给我们带来的情感和精神上的创伤，甚至比肉体的伤害更令人痛苦。

两个互相喜欢的人为什么总是吵架，亲人之间为什么聊几句就开始厌烦，出于好意却总是伤害了对方......问题的根源也许不在于人、不在于事，而在于怎么说。

1、非暴力沟通的四个要素

1.1 观察而非评论

非暴力沟通的第一个要素是观察。我们仔细观察正在发生的事情，并清楚地说出观察结果。非暴力沟通并不要求我们保持完全的客观而不作任何评论。它只是强调区分观察和评论的重要性。将观察和评论混为一谈，人们将倾向于听到批评，甚至会产生逆反心理。

就以“你心里没我”这句话为例，这就不是观察，而是评论。观察应该是一个客观事实，比如“你今天一天都没跟我说话”。其他如批评、指责、辱骂、归类、比较都是一种评论，如果再配合上“每次”、“总是”、“从不”这样的词，更可能会招来逆反。

其实我们对他人的评论反应的是我们自己的需要，我们可能只是在一件事上没有得到自己想要的，却以评论的方式完全否定了对方的所有。要做到非暴力沟通，就需要先做到清楚的在特定的时间和情境中观察，先不对观察到的进行评论。

1.2 感受而非想法

非暴力沟通的第二个要素是感受。当我们通过观察，知道了客观事实，此时我们对客观事实的感受是怎样的，比如高兴、紧张、放松等。清楚的表达感受需要丰富的词汇，而不仅仅是通过“好”、“一般”、“不好”来形容，这类的词过于的主观，不同的人对不同的事，对好的定义都不是相通的。而“欣喜”、“高兴”、“兴奋”、“喜出望外”、“满足”这类的词可以表达不同层级的感受。

想法与感受的区别是什么？想法类似于对自己的感受做评价。“我考试不及格，真沮丧”是感受；“我考试不及格，我觉得自己真差劲”是想法。

1.3 感受自己的需要

非暴力沟通的第三个要素是需要。我们生气时，表面上看因为对方的行为，导致了我们的感受，而从根源上看，是因为对方的行为，导致了我们的需要没有被满足，最终导致了我们的感受。很多负面情绪都暗含着期待，间接的表达着我们尚未满足的需要。

以“你心里没我”这句话为例，女孩实际上是渴望得到关心。可通过批评的方式来提，男孩的反应通常是辩解或反击。其实女孩直接说出自己的需要，更可能得到积极的回应。但我们更习惯于考虑别人有什么错。

当感受到对方的负面情绪时，我们有四种选择：

  1. 认为自己犯了错。

  2. 指责对方。

  3. 了解我们自己的感受和需要。

  4. 用心体会他人的感受和需要。

前两种选择都只是情绪的宣泄，双方的需要都不会被满足。只有了解了感受和需要，才可能真正的去解决冲突。

1.4 请求而非命令

非暴力沟通的第四个要素是请求。我们感受到了自己真正的需要，就可以将它通过正确的方式传达给对方。我们要清楚的告诉对方，我们希望对方做什么，而不是别做什么。

女孩生气的对男孩说“你少花点时间在游戏上”，男孩就“听话”的去看小说了，结果女孩还是生气。她想当然的认为自己表达了不开心的情绪，男孩应该知道她的需要是多陪她，而不是去把时间花在其他事情上。

我们的意思和别人的理解有时可能是两回事。如果无法确定对方是否已经明白，我们可能就需要得到反馈。请求反馈能确保对方准确把握我们的意思。

提出请求时最要注意的就是请求变成了命令。一旦对方认为不答应我们就会受到责罚，对方就会把我们的请求看作是命令。听到命令时，一个人只能看到两种选择：服从或反抗。不论如何，只要对方认为我们是在强迫，对方就不会乐于满足我们的需要。如果我们在过去常常指责他人，我们的请求很可能就会被看作是命令。而一个经常受到指责的人也会倾向于将请求解读为命令。

2、先听对方讲完

很多时候我们为了表示自己的关心，对方在难过时，我们就总想着得做点什么，说点安慰的话，给出行动建议，讲讲相同的经历......这些其实都妨碍了对方的倾诉，我们的倾听。

倾听时我们要放下已有的想法和判断，不论对方以什么样的方式来表达自己，我们首先要做的是用心体会对方的感受和需要，而不是急于表达我们的态度和感受。

妨碍倾听和体会他人处境的几种常见行为：

  • 建议：“我想你应该……”

  • 比较：“这算不了什么。你听听我的经历……”

  • 说教：“如果你这样做……你将会得到很大的好处。”

  • 安慰：“这不是你的错；你已经尽最大努力了。”

  • 回忆：“这让我想起……”

  • 否定：“高兴一点。不要这么难过。”

  • 同情：“哦，你这可怜的人……”

  • 询问：“这种情况是什么时候开始的？”

  • 辩解：“我原想早点打电话给你，但昨晚……”

  • 纠正：“事情的经过不是那样的。”

在倾听对方的观察、感受、需要和请求之后，我们可以主动表达我们的理解。如果我们已经准确领会了他们的意思，我们的反馈将帮助他们意识到这一点。反之，如果我们的理解还不到位，他们也就有机会来纠正我们。非暴力沟通建议我们使用疑问句来给予他人反馈。这将便于他人对我们的理解作出必要的补充。

3、充分的表达愤怒

愤怒是我们的思维方式造成的，它的核心是尚未满足的需要。为了充分表达愤怒，我们有必要明了自己的需要并采取行动。但愤怒驱使我们去惩罚他人，而不是去满足需要。我们可以通过练习去免受愤怒的驱使，我们要有意识地用“我生气是因为我需要……”来取代“我生气是因为他们……”。

充分表达愤怒的第一步是我们不再归咎于他人。我们生气并不取决于他人的行为，生气的原因在于我们的想法——对他人的评判和指责。我们选择“指责他人”，我们认为别人应当认错或受罚，这就是我们生气的原因。

当我们选择第三种“了解我们自己的感受和需要”时，我们用心体会着自己，而不再分析别人犯了什么错。我们越是用心体会自己的感受和需要，我们也就越能留意到自己的心理活动。希望他人因为内疚发生改变，就是将刺激和原因混为一谈。

表达愤怒的四个步骤：

  1. 停下来。呼吸。

  2. 留意我们的指责。

  3. 体会我们的需要。

  4. 表达感受和尚未满足的需要。

4、正确的表达感激

人人都喜欢听好话，但好话也有可能造成隔阂。比如老板第一次跟你说“你辛苦了”，你可能还挺感动。可如果他一边说一边让你加班，这句话听着就很反感了。使用非暴力沟通的方式表达感激，只是为了感谢他人的行为，而不是想得到任何回报。

非暴力沟通表达感激的方式：

  1. 对方做了什么事

  2. 我们哪些需要得到了满足

  3. 我们的感受如何

我们可以站在接收人的角度想，如果对方对我说“我从你那学到了很多”，如果没有明确的上下文联系，我会觉得莫名其妙，甚至怀疑这是不是反话。而如果对方以上面的方式表达感激，明确的告诉我我做了什么，他的什么需要得到了满足，以及他的感受如何。我会毫不怀疑的接受这个感激，并且可能会一直记着。

5、总结

读这本书之前，我以为是一本俗气的《说话之道》。而读到一半就发现，我经常以“暴力”的方式与别人沟通，而别人也在以“暴力”的方式与我沟通。这本书最重要的核心不是教人如何说话，而是一种突破个人常规的思维方式。

非暴力沟通指导我们转变思维，不再条件反射式的做反应，而是去了解自己的观察、感受和需要，有意识的控制我们的语言。做到既诚实、清晰的表达自己，又尊重和倾听他人。
